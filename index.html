FAQS

  • What is Frontend Mentor, and how will it help me?

    Frontend Mentor cung cấp các thử thách code thực tế giúp bạn luyện HTML, CSS và JavaScript. Nhờ đó bạn nâng cao kỹ năng frontend, có sản phẩm sát với dự án thực tế, và xây dựng portfolio chuyên nghiệp.

  • Is Frontend Mentor free?

    Có. Frontend Mentor có gói miễn phí với nhiều thử thách. Ngoài ra có gói PRO cung cấp thêm thiết kế Figma và tính năng hỗ trợ nâng cao.

  • Can I use Frontend Mentor projects in my portfolio?

    Hoàn toàn được. Bạn có thể đưa sản phẩm mình làm từ thử thách vào portfolio để thể hiện kỹ năng và quy trình phát triển.

  • How can I get help if I'm stuck on a challenge?

    Bạn có thể tham gia cộng đồng Frontend Mentor, đặt câu hỏi trong kênh #help, hoặc đọc phần gợi ý/tài liệu đi kèm thử thách.
